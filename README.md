# Task Manager CLI

## 📖 About the Project
A command-line task management system developed in Python using Object-Oriented Programming and JSON for data persistence.

This project was built to practice clean code organization, data manipulation, and software structuring principles.

## 🚀 Technologies Used
- Python
- JSON
- Object-Oriented Programming (OOP)

## ⚙️ Features
- Add new tasks
- List all tasks
- Mark tasks as completed
- Persistent storage using JSON files

## 🧠 Concepts Applied
- Classes and Objects
- Data Structures (lists and dictionaries)
- Exception Handling
- File manipulation
- Modular programming
